Мен туралы

Мен Берікбай Томирис.Алматы қаласында 2006 жылы 5 қыркүйекте дүниеге келдім.Қазір мен Қазақ ұлттық педогогикалық университеті Абай атындағы университеттің физика мамандығы бойынша 1-курста оқып жатырмын. Мен бос уақытымды пайдалы іспен айналысқанды жақсы көремін. Менің ең үлкен қызығушылығымның бірі - математика. Бұл ғылым маған сандық әлем мен абстрактілі ойлаудың шекараларын кеңейтуге мүмкіндік береді. Математика арқылы мен логикалық ойлау қабілетімді дамытып, күрделі мәселелерді шешу дағдыларын жетілдіріп жатырмын. Есептерді шешкенде немесе жаңа теоремаларды зерттегенде, мен терең әрі жүйелі ойлауға үйренемін. Сонымен қатар, математика басқа ғылымдармен тығыз байланысты, сондықтан оның негіздерін түсіну маған физика, экономика және инженерия салаларында да қолдануға мүмкіндік береді. Математикадағы әрбір жаңалық мені жаңа мүмкіндіктер мен шешімдерге жетелейді. Мен үшін бұл ғылым тек қана теориялық білім ғана емес, өмірдің әртүрлі аспектілерінде қолданылатын құнды құрал. Математикалық ойлауды дамыту менің шығармашылық қабілеттерімді де арттырады. Дорама мен аниме – менің қызығушылығымның тағы бір ерекше бөлігі. Олар мен үшін шығармашылық пен мәдени әртүрліліктің керемет көзі. Әр дорама мен аниме өз тарихымен, кейіпкерлерімен және эмоцияларымен ерекше, сол арқылы жаңа әлемдерге саяхат жасаймын. Мен серуендеуді қатты жақсы көремін. Табиғат аясында уақыт өткізу, таза ауамен тыныстау, айналамдағы сұлулықты тамашалау мен үшін ерекше рахат сыйлайды. Бұл менің ойымды тыныштандырып, энергия береді.

"Өмір — бұл өнер, ал уақыт — ең үлкен байлық."

Іздеу

Сайтты қалай жасадым?

Бұл сайтты жасау үшін мен келесі қадамдарды орындадым:

  1. HTML көмегімен сайттың негізгі құрылымын жаздым. Бөлімдер мен элементтерді логикалық түрде орналастырдым.
  2. CSS арқылы сайт дизайнын жасадым. Әдемі түстер палитрасын таңдадым (фиолетовый, қара, қызыл).
  3. JavaScript кодын қосып, беттер арасында ауысу функциясын қостым.
  4. Жергілікті компьютерде барлық файлдарды дайындап болған соң, GitHub репозиторийін құрдым.
  5. Git командаларын қолданып, кодты GitHub-қа жүктедім.
  6. GitHub Pages арқылы сайтты жарияладым. Сайтты Интернетте көруге болатын сілтемені алдым.

GitHub арқылы сайтты қалай жарияладым?

GitHub Pages арқылы сайтты жариялау үшін келесі әрекеттерді жасадым:

  • GitHub-та жаңа репозиторий құрдым. Репозиторийдің атын `portfolio-site` деп атадым.
  • Компьютерде жазылған HTML, CSS және JavaScript файлдарын репозиторийге жүктедім:
    • `git init` — Жаңа Git репозиторийін бастадым.
    • `git add .` — Барлық файлдарды индексқа қостым.
    • `git commit -m "Initial commit"` — Файлдарды сақтау үшін commit жасадым.
    • `git branch -M main` — Негізгі тармақты (`main`) орнаттым.
    • `git remote add origin <репозиторий сілтемесі>` — GitHub репозиторийін қостым.
    • `git push -u origin main` — Файлдарды GitHub-қа жүктедім.
  • GitHub репозиторийінің баптауларынан (Settings) **Pages** бөлімін таңдадым.
  • `Deploy from branch` опциясын таңдап, негізгі тармақты (main) белгіледім.
  • GitHub автоматты түрде сайтты жариялап, маған сілтеме берді (мысалы: `https://username.github.io/portfolio-site`).
